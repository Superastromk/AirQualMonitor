# AirQualMonitor
A device that uses the Arduino Uno to read air quality, display readings on a small screen, and log data. My version has a 3d printed enclosure with a grill pattern and magnetic wall/fridge mount bracket.
# Wiring
Due to limitations in the Wokwi Software, please keep this in mind: the right side potentiometer is the MQ-135 Module and the DHT22 module is the DHT11.
